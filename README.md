# VoiceCeleb - AI Voice Cloning SaaS

A high-end, minimalist landing page for an AI Voice Cloning & Dubbing startup.

## Features

- **High-End Aesthetic**: Deep navy theme with electric blue/violet accents and glassmorphism.
- **Bilingual Support**: Instant toggling between English (EN) and Korean (KR).
- **Interactive Playground**:
  - **TTS**: Text-to-Speech simulation with waveform visualization.
  - **Dubbing**: Video/Audio file upload interface.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

```bash
npm install
```

### Running the Development Server

To start the project locally:no

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```
